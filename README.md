> weixin api 


## Getting Started

Install the module with: `npm install weixin-node`

```js
var weixin-node = require('weixin-node');
weixin-node.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g weixin-node
$ weixin-node --help
$ weixin-node --version
```
